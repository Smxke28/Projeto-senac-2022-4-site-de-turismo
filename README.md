#  Site de Turismo — Ibitipoca

Site informativo sobre o Parque Estadual do Ibitipoca, desenvolvido em HTML e CSS durante o curso de Programador de Sistemas no SENAC em 2022.

##  Páginas

| Página | Conteúdo |
|--------|----------|
| `Ibitipoca.html` | Apresentação do parque |
| `Caminhadas.html` | Trilhas e caminhadas disponíveis |
| `Pousadas.html` | Hospedagem na região |
| `Pontos.html` | Pontos turísticos |
| `Contatos.html` | Informações de contato |

##  Como usar

1. Clone o repositório:
```bash
git clone https://github.com/Smxke28/Projeto-senac-2022-4-site-de-turismo.git
```
2. Abra `Ibitipoca.html` no navegador.

##  Sobre o Parque

O Parque Estadual do Ibitipoca fica em Lima Duarte, Minas Gerais, com 1.488 hectares de área preservada. Criado em 1973 e administrado pelo IEF-MG, recebe até 800 visitantes por dia nos fins de semana.
